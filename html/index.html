Sudoku Spectrum

sudoku rules: in every row, column need the number 1-9 appear exactly once. There is only one right solution to a sudoku.
buttons:
  • undo: remove the last number you have written inside a cell (solving the grid can not be undone, only reloading will restart the game).
  • check: when you have completely filled the grid with numbers you can use this button to check the grid.
  • clear: clear the currently displayed grid to an empty grid. You can then write your own sudoku puzzles for expample.
  • solve: clicking this button will trigger an algorithm, wich will try and solve the currently displayed sudoku. (hint: Sadly, the algorithm is not yet powerful enough to solve every medium and hard sudoku)
  • download: download the currently displayed sudoku grid
